# Full Course Outline: Build a Trading Bot with Alpaca API from Scratch

This document outlines a complete, 30-part tutorial series designed to guide a user from absolute beginner to a confident bot developer. The course is structured into six modules, each building upon the last.

---

## Module 1: Foundations & First Steps (Tutorials 1-5)

**Goal:** To set up the development environment and perform the first simple, successful action with the API.

| No. | Tutorial Title | What You'll Learn |
|:---:|:---|:---|
| 1. | **Your First Step – Generating API Keys in Alpaca** | How to create an account and generate API keys for the paper trading environment. |
| 2. | **"Hello, Alpaca!" – How to Connect & Check Your Account Status** | How to write your first Python script to connect to the API and print your account equity. |
| 3. | **Fetching Market Data – Your First Candlestick** | How to retrieve historical price data (OHLC) for a specific symbol (e.g., AAPL). |
| 4. | **Your First Trade – Placing a `MARKET` Order** | How to write a script that buys or sells a single share of a stock at the current market price. |
| 5. | **What Happened to My Order? – Checking Order Status & Positions** | How to confirm if an order has been filled and check your current open positions. |

---

## Module 2: Bot Logic & Simple Strategies (Tutorials 6-10)

**Goal:** To understand the core loop of a bot and implement a first, basic trading strategy.

| No. | Tutorial Title | What You'll Learn |
|:---:|:---|:---|
| 6. | **Anatomy of a Bot – What Is a Main Loop and How to Build It?** | How to create an infinite loop that will serve as the bot's "heartbeat," executing actions at regular intervals. |
| 7. | **Calculating Your First Indicator – Simple Moving Average (SMA)** | How to use the `pandas` library to calculate a simple moving average from recent price data. |
| 8. | **Simple Entry Logic – Detecting a Moving Average Crossover** | How to write an `if` statement that detects the moment two moving averages cross. |
| 9. | **Simple Exit Logic – Selling on a Reversal Signal** | How to implement the exit condition for when the trend signal reverses. |
| 10.| **Your First Complete Bot – Putting It All Together** | How to combine the loop, indicators, and logic into a single, working MA Crossover bot prototype. |

---

## Module 3: Risk Management – The Holy Grail of Trading (Tutorials 11-15)

**Goal:** To implement the crucial mechanisms that protect trading capital.

| No. | Tutorial Title | What You'll Learn |
|:---:|:---|:---|
| 11.| **The Safety Net – How to Set a Simple Stop-Loss?** | How to submit bracket orders that automatically attach a Stop-Loss and Take-Profit to your entry. |
| 12.| **Take Your Profits! – Setting a Take-Profit Order** | How to precisely define the profit target for your bracket order. |
| 13.| **Never Risk Too Much – Calculating Position Size** | How to dynamically calculate the number of shares to buy to risk a fixed percentage of your capital (e.g., 1%). |
| 14.| **Protecting Your Profits – What is a Trailing Stop-Loss?** | A conceptual explanation of how a trailing stop-loss works to lock in profits. |
| 15.| **Implementing a Trailing Stop-Loss in Code** | How to write the logic to manually manage and update a stop-loss order as a trade moves in your favor. |

---

## Module 4: Advanced Indicators & Strategies (Tutorials 16-20)

**Goal:** To expand the bot's strategic arsenal with popular and effective techniques.

| No. | Tutorial Title | What You'll Learn |
|:---:|:---|:---|
| 16.| **Relative Strength – Building an RSI-Based Bot** | How to use the Relative Strength Index (RSI) to trade overbought and oversold conditions. |
| 17.| **The Magic of Volatility – A Bollinger Bands® Bot** | How to trade breakouts from a low-volatility "squeeze" using Bollinger Bands. |
| 18.| **The Power of Momentum – Implementing a MACD Strategy** | How to use MACD line and signal line crossovers to generate trading signals. |
| 19.| **Signal Confirmation – Combining Two Different Indicators** | How to build a bot that only enters a trade when both RSI and MACD give a confirmation signal. |
| 20.| **The Bigger Picture – How to Analyze Multiple Timeframes** | How to check the primary trend on a daily chart (D1) and look for entry signals on an hourly chart (H1). |

---

## Module 5: Professional Bot Architecture (Tutorials 21-25)

**Goal:** To transform a simple script into a robust and professional piece of software.

| No. | Tutorial Title | What You'll Learn |
|:---:|:---|:---|
| 21.| **The Bot's "Black Box" – Logging Every Decision to a File** | How to use Python's `logging` library to save all actions and errors to a `.log` file for later review. |
| 22.| **Stop Digging in the Code – Using an External Config File** | How to move all settings (API keys, strategy parameters) to a `config.json` file for easy changes. |
| 23.| **What If the Bot Restarts? – Managing Position State** | How to save the bot's current state (e.g., `is_in_position`) to a file so it can recover after a restart. |
| 24.| **Stay Updated – Sending Real-Time Notifications to Discord** | How to use webhooks to send instant notifications about your trades to a private Discord channel. |
| 25.| **Making Your Bot Resilient – Handling API and Connection Errors** | How to use `try-except` blocks to ensure your bot survives temporary network issues or API failures. |

## Module 6: Testing, Optimization & Deployment (Tutorials 26-30)

**Goal:** To validate the strategy on historical data and launch the bot into a live environment.

| No. | Tutorial Title | What You'll Learn |
|:---:|:---|:---|
